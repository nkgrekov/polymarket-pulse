Commands

This is the full command reference. If you want the scenario-first explanation, open “How it works?”.

Core commands

/start

Activate profile and get quick-start flow.

/menu

Open inline one-tap action menu.

/help

Show core + advanced command reference.

/movers

Show top live movers with fallback windows.

/watchlist

Show live watchlist changes for your profile.

/inbox

Show latest alert inbox rows (up to 10).

/plan

Show current plan, usage, and upgrade next step.

/upgrade

Open PRO offer and instant Stars invoice.

Advanced commands

/threshold 0.03

Set personal minimum delta for alert filtering.

/watchlist_add <market_id|slug>

Add market manually or open picker when argument is empty.

/watchlist_remove <market_id|slug>

Remove market from your watchlist.

/watchlist_list

List current watchlist markets.

/limits

Display FREE vs PRO limits and current usage.

/inbox20

Extended inbox output (up to 20 rows).

Restricted

/admin_stats

Admin-only aggregate stats (requires allowlist).

Commands

Это полный справочник команд. Если нужен сценарный формат, откройте “Как это работает?”.

Как это работает? -> /how-it-works Открыть Telegram-бота

Core команды

/start

Активация профиля и быстрый старт.

/menu

Inline-меню one-tap действий.

/help

Список core + advanced команд.

/movers

Top live movers с fallback по окнам.

/watchlist

Live-изменения по вашему watchlist.

/inbox

Последние alert-строки (до 10).

/plan

Текущий план, usage и шаг к апгрейду.

/upgrade

PRO-оффер и мгновенный инвойс Stars.

Advanced команды

/threshold 0.03

Персональный порог фильтрации сигналов.

/watchlist_add <market_id|slug>

Добавление рынка вручную или запуск picker без аргумента.

/watchlist_remove <market_id|slug>

Удаление рынка из watchlist.

/watchlist_list

Список рынков в watchlist.

/limits

Лимиты FREE/PRO и текущий usage.

/inbox20

Расширенный inbox (до 20 строк).

Ограниченный доступ

/admin_stats

Админская сводка (требуется allowlist).

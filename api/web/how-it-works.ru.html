Открыть Telegram-бота Полный список команд -> /commands

Только открыл бота первый раз

/start

Бот активирует профиль, показывает быстрые действия и самый короткий путь к первому сигналу.

Профиль активирован
Быстрый старт:
1) /menu
2) /movers
3) /watchlist
4) /threshold 0.03

Хочу быстро увидеть что движется

/movers

Бот показывает топ-3 рынков по сдвигу вероятности в текущем live-окне и включает fallback на 30m/1h, если окно плоское.

Trump 2028      34% -> 41%   ↑7пп  · 11м назад
Recession 2026  48% -> 44%   ↓4пп  · 39м назад
Fed Cuts July   29% -> 36%   ↑7пп  · 1ч назад

Хочу следить за конкретными рынками

/watchlist_add · /watchlist_list · /watchlist_remove · /watchlist

Добавляете через picker или slug/market_id, проверяете текущий список, убираете неактуальные рынки и смотрите только активный набор.

/watchlist_add
-> picker с фильтрами (All/Crypto/Politics/Macro)

/watchlist_list
-> текущие рынки в вашем watchlist

/watchlist_remove <market_id|slug>
-> удалить один рынок из watchlist

/watchlist
-> live-изменения выбранных рынков
-> fallback 30m/1h при плоском latest

Хочу настроить чувствительность алертов

/threshold

Это ваш персональный минимальный размер движения. Всё ниже порога не попадает в inbox и push.

/threshold 0.03
-> Порог обновлен: 0.030

Совет: 0.02 = чувствительнее,
выше 0.03 = строже фильтрация шума.

Хочу проверить, что пропустил

/inbox · /inbox20

Inbox показывает последние кандидаты сигналов для вашего профиля по убыванию абсолютной дельты и с учетом threshold.

/inbox
-> последние алерты (до 10)

/inbox20
-> расширенный просмотр (до 20)

Если пусто, бот объяснит: это порог или плоский рынок.
